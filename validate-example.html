Checking Your Network#

Selecting which rates to include and which to exclude from your network is a bit of an art. pynucastro has a few tools to help check what you might be missing.

import pynucastro as pyna

reaclib_library = pyna.ReacLibLibrary()

RateCollection validate method#

Let‚Äôs start by trying to create a network for carbon burning.

To start, let‚Äôs pick the nuclei \(\alpha\), \({}^{12}\mathrm{C}\) and \({}^{20}\mathrm{Ne}\)

nuclei = ["he4", "c12", "ne20"]
cburn_library = reaclib_library.linking_nuclei(nuclei)

Now we can make a rate collection

rc = pyna.RateCollection(libraries=[cburn_library])
rc

C12 ‚ü∂ He4 + He4 + He4
C12 + C12 ‚ü∂ He4 + Ne20
Ne20 + He4 ‚ü∂ C12 + C12
He4 + He4 + He4 ‚ü∂ C12 + ùõæ

Now, since we are primarily interested in \({}^{12}\mathrm{C} + {}^{12}\mathrm{C}\), let‚Äôs make sure we are not missing any other reactions that have the same reactants. The validate() method will do this, by comparing the rates we have selected to another library.

rc.validate(reaclib_library)

validation: Ne20 produced in C12 + C12 ‚ü∂ He4 + Ne20 never consumed.

validation: missing He4 + He4 + He4 ‚ü∂ p + B11 as alternative to He4 + He4 + He4 ‚ü∂ C12 + ùõæ (Q = -8.682 MeV).
validation: missing He4 + He4 + He4 ‚ü∂ n + C11 as alternative to He4 + He4 + He4 ‚ü∂ C12 + ùõæ (Q = -11.4466 MeV).
validation: missing C12 + C12 ‚ü∂ p + Na23 as alternative to C12 + C12 ‚ü∂ He4 + Ne20 (Q = 2.242 MeV).
validation: missing C12 + C12 ‚ü∂ n + Mg23 as alternative to C12 + C12 ‚ü∂ He4 + Ne20 (Q = -2.598 MeV).

False

This tells us that we are missing 2 branches of the \({}^{12}\mathrm{C} + {}^{12}\mathrm{C}\) reaction. ReacLib already scales the rates based on the branching of the products, so we should try to include these other branches.

Note

By default, validate() only checks forward rates.

To address these issues, we need to include the additional nuclei. In particular, the branch that makes \({}^{23}\mathrm{Na}\) is likely important (the rate making \({}^{23}\mathrm{Mg}\) is endothermic, so less likely).

nuclei += ["p", "na23"]
cburn_library = reaclib_library.linking_nuclei(nuclei)
rc = pyna.RateCollection(libraries=[cburn_library])
rc

nuclei += ["mg24"]
cburn_library = reaclib_library.linking_nuclei(nuclei)
rc = pyna.RateCollection(libraries=[cburn_library])
rc
